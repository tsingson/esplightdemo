# ESP32 led blink demo

ESP32 dev module with LED blink / on / off , default is blink
control via Golang cli throut serial port , code in cmd/cli
```
 ./cmd/bin/cli ?

? should be 0 / 1 / 2 / 9 
```
control via BlueTeeth (BLE) via go cli , code in cmd/ble

```
 ./cmd/bin/ble ?

? should be 0 / 1 / 2 / 9 
```
